## 07. CONCATINATION operator(||)

- To add or join two string expressions
 

      Example: 
      
        SQL> SELECT 'WELCOME'||'TO ORACLE' FROM DUAL;

        'WELCOME'||'TOOR
        ----------------
        WELCOMETO ORACLE

        SQL> SELECT 'WELCOME'||' '||'TO ORACLE' FROM DUAL;

        'WELCOME'||''||'T
        -----------------
        WELCOME TO ORACLE

        SQL>
